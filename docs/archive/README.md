# Archive

Historical planning docs, completion reports, fix summaries, status reports, and handoff notes from past work. Kept for reference but not actively maintained.

If you're looking for current information:

- Project conventions for AI assistants → `/CLAUDE.md`
- Operational setup guides → `/*_SETUP_GUIDE.md`, `/*_QUICK_REFERENCE.md` at repo root
- Database state → `/database-migrations/` (migrations are authoritative)
- Recent feature work → git history (`git log`) is more reliable than these docs

If you find yourself referencing one of these archived docs frequently, that's a signal it should be promoted back to root and updated.
